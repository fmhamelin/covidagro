Introduction à la modélisation en épidémiologie : COVID19 - 1er confinement - estimation de la reproductivité du virus

Cécile Berthelot, Alizée Delarue, Aglaé Thieffry, Amélie Picchiottino (M1 Agro Rennes), le 15/03/2021.

Edité par Frédéric Hamelin, le 25/03/2021.

Introduction

Nous nous sommes intéressées au cours de ce projet à la modélisation de épidémie de Covid-19 lors du premier confinement en France, à partir du 19 mars (par manque de données les jours précédents).

Pour cela, nous avons récolté des bases de données sur data.gouv.fr. Nous avons trouvé des données portant sur le nombre de tests, le nombre de tests positifs, le nombre de décès, le nombre de guérisons. L’ensemble des données a été récolté dans les hôpitaux, nous n’avons donc pas de données sur les guérisons et les décès pour les personnes restées isolées chez elles. Les données sont cumulées à partir du 19 mars.

Sources: les données utilisées viennent de data.gouv.fr

https://www.data.gouv.fr/fr/datasets/donnees-hospitalieres-relatives-a-lepidemie-de-covid-19/

https://www.data.gouv.fr/fr/datasets/donnees-relatives-aux-resultats-des-tests-virologiques-covid-19/

Notre étude s’appuie sur les travaux sur le confinement réalisés par Roques et al, 2020 :

Roques L, Klein EK, Papaïx J, Sar A, and Soubeyrand S (2020) Impact of Lockdown on the Epidemic Dynamics of COVID-19 in France. Front. Med. 7:274.

Modèle SIRD

Nous définissons les paramètres de notre modèle :

N = 67e6 # Taille totale de la population
alpha = 1/10 # Taux de guérison (par jour) 
R_0 = 3 # Reproductivité du virus 
beta = alpha*R_0 # R0 = beta/alpha
sigma = 0.7 # Sensibilité des tests PCR
kappa = 5e-3 # Biais d'observation
gamma = 0.01 # Taux de mortalité
t0 = 1 # Correspond au 1er avril, la date du début de l'étude
theta0 = c(beta,kappa,gamma) # Vecteur des paramètres à estimer

Nous définissons les conditions initiales du modèle :

I0 = 1.5e6 # On fixe le nombre initiale d'infectés
R0 = N*0.01 # Inconnue : on suppose qu'au 1er avril il y avait 1% d'immunisés
D0 = DN[1]  # On se sert de la donnée initiale
S0 = N-I0-R0-D0
X0 = c(S0,I0,D0) # Vecteur d'état

Le temps est défini ci dessous :

t = t0:lt

La fonction du modèle SIRD :

SIRD = function(t, X, P){ # X : vecteur d'état; P : paramètres du modèle
  beta = P[1] ; gamma = P[2]
  S = X[1] ; I = X[2] ; D = X[3]
  y = beta*S*I/N # Nombre de nouvelles infections par jour
  dS = -y
  dI = +y - alpha*I - gamma*I
  dD = +gamma*I
  dX = c(dS,dI,dD) # Variations de S, I et D mises à jour 
  return(list(dX))
}

La fonction qui calcule la vraisemblance des paramètres est définie ci-dessous :

logLike = function(theta){ # theta : vecteur des paramètres à estimer
  beta = theta[1]
  kappa = theta[2]
  gamma = theta[3]
  P = c(beta,gamma)
  X = ode(X0,t,SIRD,P) # X : vecteur des solutions du modèle SIRD
  p1 = sigma*X[,3]/(X[,3]+kappa*X[,2]) # Proportion théorique de tests quotidiens positifs
  L = dbinom(pt, tt, p1, log=TRUE) + dpois(dn, gamma*X[,3], log=TRUE) #  Vraisemblance : calcule la probabilité d'observer pt en connaissant la probabilité théorique p1 qu'un test soit positif (loi binomiale) et d'observer dn en connaissant la quantité théorique quotidienne de mort basée sur le taux de mortalité gamma (loi de poisson)
  LL = sum(L)
  return(LL)
}

Nous cherchons ensuite à estimer et optimiser les paramètres beta, kappa et gamma de notre modèle à partir des conditions initiales (theta0) à l’aide d’une fonction optimisant la vraisemblance (logLike) :

opt = optim(theta0,logLike,control=list(fnscale=-1))

Nous utilisons ensuite nos paramètres optimisés pour résoudre notre système d’équations différentielles.

t = t0:50

X = ode(X0,t,SIRD,P) # X : vecteur des solutions du modèle SIRD avec les paramètres optimisés

A l’aide de nos paramètres estimés, nous sommes également en mesure de calculer le nombre de reproduction effectif du virus au cours du confinement :

Re = beta/(alpha+gamma) # Re : nombre de reproduction effectif

## [1] "Re= 0.439628746558056"
